# AIChatContextQuestions

**Интеллектуальный опросник для SillyTavern**

Это расширение перехватывает каждое сообщение пользователя, отправляет его в текущую LLM‑модель SillyTavern для получения трёх уточняющих вопросов, отображает их в модальном окне, сохраняет ответы и внедряет их в окончательный системный промпт.

## Установка
1. Скопируйте папку `AIChatContextQuestions` в директорию `public/extensions` вашего SillyTavern.
2. Перезапустите SillyTavern или нажмите **Reload extensions** в UI.

## Файлы
- `extensionManifest.json` – метаданные расширения.
- `index.js` – основной скрипт (логика скрытого запроса, UI, hooks).
- `README.md` – этот файл.
- `.gitignore` – исключает ненужные файлы из репозитория.

## Как работает
- **onUserMessage** – после отправки сообщения отображается оверлей "ИИ анализирует сообщение…", затем вызывается скрытый запрос `sillytavern.generateRaw`. Если запрос не удался, используются fallback‑вопросы.
- **onPromptGeneration** – перед генерацией ответа ответы берутся из `sessionStorage`, формируется блок `[User Answers to Contextual Questions]` и добавляется к промпту.

## Безопасность
- Расширение использует только публичные API SillyTavern, не делает запросов к внешним сервисам.
- Ответы хранятся в `sessionStorage` и очищаются после инъекции.
